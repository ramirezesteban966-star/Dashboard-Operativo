Cómo obtener la URL del CSV:
1. Abrí tu Google Sheet → Archivo → Compartir → Publicar en la web
2. Elegí la hoja "Datos Detallados" → Formato: Valores separados por comas (.csv) → Publicar
3. Copiá la URL y pegala arriba → Cargar datos
Primero ejecutá el script de extracción en Apps Script para poblar el Google Sheet, luego publicalo como CSV y pegá la URL acá.
